📊Commit Chronicle

Telegram-бот для глубокого анализа активности GitHub-аккаунтов. По одной команде собирает коммиты, формирует сводку и генерирует ультра-сжатый JSON-файл для последующей передачи в LLM.

🔮Tarot Bot

Асинхронный бот для гадания на картах Таро с интегрированным AI-Оракулом, интерпретирующим расклады на основе контекста и настроения пользователя.

  • Паттерн Circuit Breaker для ИИ: автоматическая блокировка запросов к API при сбоях провайдера и переключение на fallback-сценарии.
  • Интеллектуальное кэширование ответов Оракула в Redis для снижения затрат на LLM.
  • Изолированный RAG-контекст: история и настроение пользователя дополняют системный промт.
  • Встроенный асинхронный бэкап базы данных (pg_dump) с распределенной блокировкой в Redis.
